# TrotBot 文档导航（根目录）

该目录是 `trotbot` 的统一文档入口，按“长期沉淀、跨模块协作”原则放在工作空间根 `docs/` 下，便于检索与团队共享。

## 核心文档

- `软件需求清单.md`：按优先级的软件需求列表与实现状态
- `架构说明.md`：当前架构（As-Is）与目标架构（To-Be）
- `开发问题跟踪.md`：开发/联调/实测问题记录与闭环模板
- `本地运行指南.md`：运行指南入口（指向包内主指南）

## 放置原则

- 根 `docs/trotbot/`：需求、架构、问题跟踪、评审类文档（协作型）
- `src/trotbot/`：与包运行强绑定、操作频繁的主说明（执行型）
- `src/trotbot/docs/images/`：图片资源（与包内 README 相对路径兼容）

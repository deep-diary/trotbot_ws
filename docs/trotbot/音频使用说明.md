# 音频使用说明（鲁班猫实机 / PulseAudio）

适用于在 **LubanCat 等 ARM 板**上通过 **ALSA + PulseAudio** 使用板载声卡与 **蓝牙音箱/耳机**。命令在板卡本地终端执行。

**前置**：已安装 `alsa-utils`、`pulseaudio`（或 PipeWire 且仍提供 `pactl`/`paplay`）；蓝牙音箱已在系统设置中配对并连接。

---

## 1. 先分清两条链路

| 目的 | 底层含义 |
|------|----------|
| **播放** | 使用 PulseAudio 的 **sink（输出）**；蓝牙音箱一般为 `bluez_sink....` |
| **录音（环境/麦克）** | 使用 **source（输入）**，如板载 `alsa_input...es8388...` 或 `arecord -Dhw:X,0`；**不是** `bluez_sink....monitor`（见下文） |

`bluez_sink.<MAC>.a2dp_sink.monitor` 只反映 **本机送往该蓝牙 sink 的数字音频**（内录），**不能**录手机播到音箱的声音，也**不能**代替音箱上的语音麦克。

---

## 2. 查找播放设备（含蓝牙音响）

列出 **Pulse 输出设备（sink）**：

```bash
pactl list short sinks
```

只看蓝牙相关：

```bash
pactl list short sinks | grep -i bluez
```

典型蓝牙音箱一行示例：

```text
4  bluez_sink.78_DA_07_B4_28_CB.a2dp_sink  module-bluez5-device.c  s16le 2ch 44100Hz  SUSPENDED
```

第二列 **`bluez_sink.xxx.a2dp_sink`** 即播放时要用到的 **设备名**（全文复制使用）。

查看 **ALSA 物理声卡**（板载 HDMI/DP/ES8388 等，**不含**蓝牙逻辑名）：

```bash
aplay -l
```

---

## 3. 查找录音设备

列出 **Pulse 输入设备（source）**：

```bash
pactl list short sources
```

板载 ES8388 麦克/线路一般类似：`alsa_input.platform-es8388-sound....source`。

查看 **仅 CAPTURE 的 ALSA 设备**（与 `hw:X,0` 对应）：

```bash
arecord -l
```

录音用的 **card 号以本机 `arecord -l` 为准**（不同镜像可能不是 `hw:2,0`）。

---

## 4. 设置默认播放 / 默认录音

**默认播放（输出）** 设为蓝牙音箱：

```bash
pactl set-default-sink bluez_sink.78_DA_07_B4_28_CB.a2dp_sink
```

将上面 sink 名换成你机器 `pactl list short sinks` 里 **带 `bluez_sink` 的那一整串**。

**默认录音（输入）** 设为板载输入示例：

```bash
pactl set-default-source alsa_input.platform-es8388-sound.HiFi__hw_rockchipes8388__source
```

将 source 名换成你机器 `pactl list short sources` 里实际需要的那一行（第二列）。

验证当前默认：

```bash
pactl info | grep -E 'Default Sink|Default Source'
```

---

## 5. 查看与调节播放音量（含蓝牙音响）

音量由 PulseAudio 按 **sink** 管理；蓝牙音箱对应 **`bluez_sink....a2dp_sink`**，与其它输出设备用法相同。

### 查看音量

**查看指定 sink 当前音量**（左右声道各一行）：

```bash
pactl get-sink-volume bluez_sink.78_DA_07_B4_28_CB.a2dp_sink
```

将 sink 名换成 `pactl list short sinks | grep bluez` 中的完整名称。

**查看该 sink 详细信息**（含音量、静音、描述等）：

```bash
pactl list sinks | grep -A 30 'Name: bluez_sink'
```

**当前默认输出是谁**（便于确认调的是否为蓝牙）：

```bash
pactl info | grep 'Default Sink'
```

### 调节音量

**设为百分比**（常用）：

```bash
pactl set-sink-volume bluez_sink.78_DA_07_B4_28_CB.a2dp_sink 80%
```

**相对当前音量增减**（例如提高 5%）：

```bash
pactl set-sink-volume bluez_sink.78_DA_07_B4_28_CB.a2dp_sink +5%
```

也可用 **分贝**，例如：

```bash
pactl set-sink-volume bluez_sink.78_DA_07_B4_28_CB.a2dp_sink -10dB
```

**静音 / 取消静音**：

```bash
pactl set-sink-mute bluez_sink.78_DA_07_B4_28_CB.a2dp_sink 1    # 静音
pactl set-sink-mute bluez_sink.78_DA_07_B4_28_CB.a2dp_sink 0    # 取消
```

若已将蓝牙设为 **默认 sink**，也可用 **`@DEFAULT_SINK@`** 代替长名称：

```bash
pactl get-sink-volume @DEFAULT_SINK@
pactl set-sink-volume @DEFAULT_SINK@ 70%
```

说明：此处调节的是 **送往蓝牙音箱的数字音量**；音箱本体若还有物理旋钮/语音音量，需另外在设备上调节。

---

## 6. 命令行播放

**指定蓝牙 sink（不依赖默认设备）**：

```bash
paplay -d bluez_sink.78_DA_07_B4_28_CB.a2dp_sink /path/to/audio.wav
```

**已设置默认 sink 为蓝牙后**，可走 Pulse 插件：

```bash
aplay -D pulse /path/to/audio.wav
```

或直接：

```bash
paplay /path/to/audio.wav
```

---

## 7. 命令行录音

**走 ALSA 硬件（常用：板载麦克）**，格式与官方文档类似：

```bash
arecord -f cd -Dhw:2,0 -d 10 /path/to/out.wav
```

其中 **`hw:2,0` 须与 `arecord -l` 中 CAPTURE 的 card/device 一致**。

**走 Pulse 默认 source**（适合已 `set-default-source` 的场景）：

```bash
arecord -f cd -D pulse -d 10 /path/to/out.wav
```

**单次命令指定 source、不改默认**：

```bash
PULSE_SOURCE=alsa_input.platform-es8388-sound.HiFi__hw_rockchipes8388__source \
  arecord -f cd -D pulse -d 10 /path/to/out.wav
```

## 8. 更换蓝牙音响后要改什么

- 需要更新的通常是 **Pulse 里的名字字符串**（例如 `bluez_sink.<蓝牙 MAC>.a2dp_sink`），**不是**固定不变的「设备号」。
- `pactl list short sinks` 里第一列的数字（如 `0`、`4`）在重启或模块加载顺序变化后**也可能变**，**不要**长期写死依赖该数字。
- **正确做法**：换新音箱或重新配对后，重新执行：

  ```bash
  pactl list short sinks | grep -i bluez
  ```

  用输出中的 **完整 sink 名** 更新 `set-default-sink`、`paplay -d` 等命令。

- 板载录音若硬件未变，`arecord -l` 仍可用来确认 `hw:X,0` 是否改变。

---

## 9. 常见问题

| 现象 | 建议 |
|------|------|
| `arecord -Dhw:0` 报错「没有那个文件或目录」 | 该 card 可能无 CAPTURE；查 `arecord -l` 换对有 `card X` 录音的设备。 |
| 蓝牙播放无声 | 确认已连接；`pactl set-sink-volume <sink名> 80%`；用 `paplay -d <sink>` 显式指定。 |
| 录 `bluez_sink...monitor` 全是静音 | 仅当 **本机** 正在往该 sink 播放时才有信号；环境声请用麦克 source / `hw:X,0`。 |
